# Contoso アウトドア製品 FAQ（サンプル）

## 保証
- 当社のテント・寝袋・バックパックの**保証期間は3年**です。通常使用での縫製・ファスナーの不具合が対象です。
- 保証を受けるには購入時のレシートまたは注文番号が必要です。

## 寝袋 CozyNights
- CozyNights 寝袋の**快適使用温度は -5℃**、限界温度は -12℃ です。
- 重量は 1.4kg、収納サイズは 38cm × 22cm です。

## テント TrailMaster
- TrailMaster は 2〜3 人用のドーム型テントで、**耐水圧は 3000mm** です。
- 設営の目安時間は 10 分、総重量は 2.8kg です。

## 返品・交換
- 未使用品は**購入から30日以内**であれば返品可能です。
- 返品送料は初期不良の場合は当社負担、それ以外はお客様負担です。
